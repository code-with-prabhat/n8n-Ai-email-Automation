## What is n8n?
n8n is a powerful, extendable workflow automation tool that allows you to connect various services and applications with ease. It provides a user-friendly interface to create workflows that automate repetitive tasks, integrate APIs, and process data efficiently. With n8n, you can build complex workflows without writing extensive code.

## Project Description
This project is an **Email Sender Automation** built using n8n. The workflow is triggered every 2 minutes and performs the following tasks:    

1. **AI-Powered Content Generation**: Utilizes the Gemini LLM (Large Language Model) to generate AI-driven content based on predefined prompts or inputs.
2. **Email Dispatch**: Sends the generated content to a specific email address automatically.

### Key Features

- **Automated Trigger**: The workflow runs every 2 minutes, ensuring timely execution.
- **AI Integration**: Leverages the Gemini LLM model for intelligent content generation.
- **Email Automation**: Seamlessly sends the AI-generated content to a designated recipient.

This project demonstrates the power of combining n8n's automation capabilities with advanced AI models to streamline email communication workflows.
